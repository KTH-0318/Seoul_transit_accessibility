# Seoul_transit_accessibility

# 접근성을 고려한 서울시 대중교통 취약지역 도출

경사도를 반영한 실제 보행 네트워크(Tobler 함수 + Dijkstra) 위에서 2SFCA로 실질 접근성을 측정하고,
사분면 분석으로 **"수요는 많은데 접근성이 낮은" 핵심 취약지역 2,211개 격자를 도출**한 공간 분석 프로젝트.

## 분석 흐름 및 파일 구성

| 단계 | 파일 | 내용 |
| --- | --- | --- |
| 1. 네트워크 구축 | `[파일명]` | 경사 반영 보행 네트워크 구축 (노드 21만 · 링크 28만, Tobler 보행 함수) |
| 2. 접근권역 산정 | `[파일명]` | 격자 12,683곳 → 정류장·역 Dijkstra 최단 보행시간 |
| 3. 접근성 측정 | `[파일명]` | 2SFCA 기반 격자별 실질 접근성 지수 산출 |
| 4. 취약지역 도출 | `[파일명]` | 사분면 분석 → 우선순위 점수 → 최우선 개선 후보지 선정 + Folium 지도 |

## 데이터

서울시 빅데이터캠퍼스 · 구글 Elevation API · SGIS
